# API-postgres-nodejs-js

A simple REST API built with Node.js, Express, and PostgreSQL (pg)
This project is intended as a base template for building scalable APIs.

## Tech Stack

- Node.js
- Express
- PostgreSQL
- dotenv
- cors


## ⚙️ Setup

### 1. Clone the repository

```bash
git clone https://github.com/jeremypss/API-postgres-nodejs-js.git
cd API-postgres-nodejs-js
```

### 2. Install dependencies

`npm install`

### 3. Configure environment variables
Create a .env file in the root directory:

```bash
PORT=3000
DB_USER=your_user
DB_HOST=localhost
DB_NAME=your_database
DB_PASSWORD=your_password
DB_PORT=5432
```

### 4. Run the server
`npm run dev`

Server will run on:
`http://localhost:3000`
